# Welcome to Physical AI Book

Learn hands-on AI development through practical examples you can run immediately. This book is designed for beginners and intermediate learners who want to understand AI by **doing**.

## What You'll Learn

- **Hands-On Skills**: Build real AI applications with working code
- **Core Concepts**: Understand AI fundamentals through practice
- **Best Practices**: Learn industry patterns and approaches
- **Modular Learning**: Each lesson is self-contained and independently testable

## Prerequisites

To get the most from this book, you should:

- Have Python 3.11+ or Node.js 18+ installed (varies by lesson)
- Be comfortable running commands in a terminal/command prompt
- Understand basic programming concepts (variables, functions, loops)
- Have curiosity about how AI works!
